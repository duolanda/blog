    基建时间轴

    duolanda 2022/7/23

    # 2003

    # 2003.3.14 北京市人民政府同意我校征地

    北京邮电大学与北京市昌平区人民政府于2003年3月14日签订了《北京邮电大学与昌平区人民政府土地划拨协议》。北京市人民政府同意我校征地1348亩用于沙河校区建设。同年6月，教育部批准同意我校建设沙河校区。

    # 2012

    # 2012.9.16 北京邮电大学沙河校区建设工程开工仪式举行

    沙河校区开工建设是北邮师生员工近10年的期盼，是我校发展历史上的里程碑，必将极大拓展学校的发展空间，是北邮加快发展的重要机遇。

    # 2013

    # 2013.10.20 第一批建筑主体结构全部封顶

    截至10月20日，沙河校区学生公寓（雁北园）、食堂及活动中心综合楼、教学实验综合楼主体结构已全部封顶

    # 2014

    # 2014.10.28 第一批建筑基本完工

    单体学生公寓（雁北园）、教学实验综合楼、食堂及活动中心均已基本完工

    # 2015

    # 2015.9.8 沙河校区迎来第一批学生

    9月8日、9日，北京邮电大学迎来了来自五湖四海的3500余名2015级本科新生。

    这些年来，尤其进入新世纪以来，北京邮电大学办学资源紧张的困难在很大程度上制约了学校的进一步发展，尤其是学科的转型、升级、换代，迫切需要我们进一步加大学校校园建设的步伐。如果要用一句话回答你的问题——沙河校园的启动对北邮究竟有什么意义，我个人认为是“里程碑式的意义”，或者说是“划时代的意义”。

    # 2015.9.10 北京邮电大学沙河校区启用仪式暨北京高科大学联盟昌平揭牌典礼隆重举行

    # 2015.11.18 沙河校区图书馆主体结构封顶

    李杰副校长一行来到封顶作业面现场，由李杰副校长郑重宣布图书馆工程正式封顶,并亲自为图书馆顶层最后一块钢筋构架浇铸混凝土。

    沙河校区图书馆，西邻食堂及活动中心综合楼，南邻教学实验综合楼，占地面积约2万平方米，总建筑面积36700平方米，框剪结构，地下1层，地上5层。设计简约明快，建成后将成为校园的一处标志性建筑，体现我校独特的人文底蕴和教学风范。

    # 2016

    # 2016.5.27 沙河校区运动场工程顺利通过竣工验收

    沙河校区运动场工程于2015年3月10日开工建设，2016年5月初完工。该工程东临北沙河东三路、南临高教园南二街。场地东西宽约150米，南北长约240米。内设400米标准田径场一块，篮球场10块，网球场和排球场各4块，105米*68米的足球场一块，还有撑杆跳场地、跳高场地、铁饼场地、铅球投掷场地、标枪场地等，能够充分满足沙河校区教学、训练、体育活动、健身等多样化的功能需求。

    # 2016.11.30 学生宿舍Ⅱ区项目建设顺利完工

    经过为期10个月的紧张施工，我校沙河校区学生宿舍Ⅱ区项目（雁南园）圆满完成学校制定的工期目标，于11月30日顺利完工并举办了完工仪式。

    # 2017

    # 2017.9 学生宿舍II区启用

    2017 级学生顺利入驻学生宿舍II区（雁南园）。

    # 2018

    # 2018.5.26 沙河校区图书馆试运行

    沙河校区图书馆完成了基本的家具安装和文献搬迁工作，已初步具备开馆试运行条件。经研究决定，沙河图书馆于2018年5月26日起面向全校师生试运行。

    # 2019

    # 2019.1.6 教室楼 S1 结构封顶

    2019年1月6日，伴随着最后一方混凝土的浇筑到位，北京邮电大学沙河校区教室楼S1工程，在基建处、监理单位的见证下正式完成主体结构封顶。

    # 2019.12.27 学生宿舍III区结构封顶

    自2019年10月8日进入主体结构施工阶段起，至2019年12月21日最后一罐混凝土浇筑完成，历时73天，在校领导的大力支持以及各部门通力合作下，北邮沙河校区一期III区学生宿舍（雁南园S6）正式封顶。

    # 2020

    # 2020.9.30 学生宿舍III区启用

    伴随着 3700 余名 2020 级本科新生的到来和入住，也意味着学生宿舍III区（雁南园S6）的正式启用。

    # 2020.9.30 教室楼 S1 实验楼 S2、S3 竣工验收

    两项工程建设规模为10000平方米、30000平方米，分别于2018年6月及2018年12月正式开工，自开工以来，各参建单位克服疫情期间工人短缺、材料供应困难等问题，完成了合同约定的施工内容。会上，勘察、设计、监理、总包等单位做了工作汇报，设计单位负责人表示，连桥设计在施工过程中技术复杂、施工难度大，因此设计根据现场问题积极做出调整，为后续施工抢出了时间。

    # 2020.12 南大门主体竣工

    # 2021

    # 2021.03 教室楼 S1 投入使用

    我校沙河校区S1教学楼于2021年春季学期投入使用，配有特色型智慧教室、互动研讨型智慧教室和通用型智慧教室，有利于进一步提高我校教学质量。

    # 2021.03 马克思学院、人文学院学院搬迁

    学校推动校区资源整合，马克思学院、人文学院搬迁至沙河校区，进驻实验楼 S2、S3。

    # 2021.6.25 三个学院楼和南区食堂全部结构封顶

    网络空间安全学院、数字媒体与设计艺术学院和现代邮政学院（自动化学院）、理学院楼项目均为地上5层，地下2层的多层建筑，共计54175平方米。北侧三个学院楼组团和南侧理学院楼结构形式均采用钢筋混凝土框架剪力墙体系。理学院楼北侧主入口面向绿化广场，主楼规整严谨，主楼外，场地西侧布置材料制备实验室。四个学院楼建成后将极大改善师生教学环境，缓解我校办学空间不足的问题。

    南区食堂项目地上3层，地下2层，共计13105平方米。项目位于核心区景观湖南侧，紧邻学生宿舍区，地上分为就餐区、售饭区、厨房区，结构形式为框架结构。建成后将为广大师生带来舒适便捷的就餐体验和宽敞宜人的交流空间。

    # 2021.8 景观湖完工

    核心区景观湖项目已完成全部景观工程施工，成为我校历史上首个湖区景观，新学期将面向师生开放。

    # 2022

    # 2022.2.27 学校签署农机试验站现址地块四方合作框架协议

    四方协议的签署将为北邮沙河校区建设展开新的局面，极大的拓展学校办学空间，推动学校高质量建设发展。北邮将充分发挥学科人才、科技文化、社会服务等方面的优势，在人才培养、科技创新、成果转化、创新创业实践、智库建设等方面，与昌平区、国机集团、北师大开展深度合作，努力实现校地校企共赢。
