    《全校通缉》总结

    duolanda 2021/11/17

    # 前期拍摄问题

    # 曝光控制

    在拍摄中，大多数时候都是光圈优先模式，但此时不能手动调整快门。后面发现，应该手动调整光圈快门，自动 ISO，这样依然可以通过曝光补偿来控制。

    # ❓快门速度和焦段

    根据一般规律来看，快门速度是帧数的二倍，在 4k 25p 的情况下，就是 1/50,但有时候这个速度并不够。

    # ❓对焦控制

    拍运动镜头的时候很难稳定对焦，更离谱的是，有时哪怕人脸很明显也跟不上。也没有找到手动切换焦点的方法，拍周炯槃雕像的时候，只能靠不停触摸屏幕把焦点切过去。没有有效监看，拍摄时和在相机上回看都无法判断是否正确对焦。

    # ❓稳定器的使用

    不同模式的区别，按住哪个键让稳定器跟着动。

    # 场记的记录

    相机上只显示是第几条片段，甚至是照片视频一起算的，实际上保存的文件名却是 C0018。这次比较巧的是文件名正好差了 301，错一位数即可，中间也没有拍摄视频或删素材，但如果编号差了很多，场记便不会那么有效。其中一个解决方法是在导入达芬奇前重新命名，或者每次拍摄前把卡格式化。

    场记应该现场记录对应的录音文件，最好能当晚确定缺失录音，尽早补录。

    # 收音

    固定镜头不专门收音用相机收音问题也不大，运动镜头会有很大风声，一定要收音。环境音非常有用。哪怕运动镜头不收音也一定要为每句对白收音。

    # 后期问题

    # 网络视频发布响度控制

    音職人から学んだ音量調整の【考え方とコツ】Premiere Pro (opens new window)

    「音频后期」影视飓风同款插件，一键提升节目音频响度 (opens new window)

    还没有在网络上看到较好的音频响度教程，只找到了一个日本人的文章，勉强作为参考：

    • 声・・・【-6db】〜【-10db】
    • BGM・・・【-30db】〜【-35db】
    • SE・・・【-8db】〜【-15db 】

    结合听感，实际上人声是【-6db】〜【-10db】，BGM 则在【-20db】〜【-25db】。

    B 站视频中那个插件还没有尝试。

    # ❓Log 调色

    因为没有拍摄 Log 所以后期空间比较小，但如果采用 Log 可能会带来更多的噪点。需要思考，PP几能够得到一个平衡，如果拍 Log 的话，怎么曝光。

    # ❓达芬奇导入的 iPhone 录音不能批量处理成双声道

    iPhone 录的声音都是单声道，达芬奇导入后需要挨个修改素材属性才行，甚至已经拖到时间线上以后，还得改时间线上的片段属性，太繁琐了。

    # Arctime 双语字幕

    Arctime 本来提供了为中文轨道建立翻译轨道的功能，但是因为没有注意到，制作中手动创建了轨道，导致中英文字幕块入点出点有时会对不上，引来很多麻烦。

    Arctime 不能直接按照上下导出双语字幕，需要每条轨道单独导出，且属性也得针对性的修改，只能分别导出到不同的文件夹中，再在非线编软件中重新调整位置。

    # 贾老师的建议

    拍楼层牌子的镜头不要那么稳，因为田禾已经跑的上气不接下气了，要晃。而且画面内信息太多，没有重点，可以用聚焦的手法。

    田禾跑的不够用力，如果要追就要真追，才能让观众产生误解。
