    《红色警戒 3 - 起义时刻》挑战模式打法

    duolanda 2021/8/28

    # 死期：警犬

    选帝国，冲兵营，两个兵营核心堵在对方兵营门口碾压，另外一个出兵。不用帝国的话也可以用 MCV 碾压。

    # 血战：弓箭少女

    选帝国，出两个兵营在两边，每个兵营最少出三个工程师，只要占领 MCV 和出兵建筑即可为所欲为，也可以备一些炮塔核心以防不测。工程师移动时不要被弓箭少女的剑雨打到，也会判定为开展。

    # 死亡迫近：VX 打击者

    选盟军，基地车 TR，用围墙将对面的 VX 防御塔围起来，兵营也拍在对面，出标枪打 VX 和拆基地。

    # 群岛防御：磁暴步兵

    选帝国，爆 VX 即可，我打的那一局对面只有第一个基地出了很多防空兵，损失了一些 VX，后面基本都没有防空。刚开始的时候可以多建几个防御塔配合武士防御步兵，否则很容易在 VX 出来前被拆家。

    # 火炮四射：波能炮

    选帝国，出迅雷，载两个工程师，一个工程师占对面门口的波能炮（能打到对面 MCV 那个），另一个工程师尽量占重工，三个都占领之后就高枕无忧了，爆 VX 或天狗都可以。

    # 箱子与红桶：多功能步兵车

    选盟军，爆步兵，可配合 MCV 碾压。

    # 卫星终结站：火炮机车

    选盟军或帝国爆步兵都可以。

    # 女人争斗：谭雅

    这关尝试过很多回，选盟军暴兵的话，先拆苏联，帝国会四处开矿，大大延长通关时间，而如果先拆帝国，苏联会发展的很恐怖；也尝试过帝国卖本，两兵营打苏联两兵营打帝国，但一是操作不过来，二是只要苏联一出镰刀，基本就打不过了；帝国三兵营分别开在苏联、盟军、帝国门口也试过，结果是三家都拆不下来。

    目前得到的最优解是，选帝国，一兵营苏联，一兵营帝国，同时 MCV 冲到苏联基地去压镰刀，拆完苏联后顺带还能占领苏联的矿场解决资金问题，帝国那边还算好压制，保证它开不起来重工即可，顺带占领医院。

    再次玩的时候，尝试很多次之后上面方法也没有成功。发现可以开局四兵营放到下方，爆兵顺带占油井、医院，家里两个电厂后卖本，先打帝国再打盟军，最后打苏联，因为人很多，就算有镰刀也不怕

    # 钢铁交响曲：钢铁浪人

    选盟军，双兵营暴兵即可，有时候会遇到下方敌人来拆家的情况，可以先打掉左上角的敌人，差不多的时候就把出兵地点设到下方，等两个敌人都打掉之后再打左侧中间的敌人。

    # 你够格吗：突袭驱逐舰

    选盟军或帝国暴步兵，期间可以占领对方的一个精英学院，进攻时再拆掉一个精英学院。

    # 卷入战火：双刃直升机

    选盟军，暴步兵，标枪比例稍微多一些，先打左上角吉尔利斯再打右下角的维拉，可以插两个塔防止打吉尔利斯时被维拉拆家。

    # 贱招：恐怖机器人

    选苏联，一开始恐怖机器人并不会主动出来，就有了 TR 的可能，TR 时稍微离远一些，先不要吸引到收割机甲的仇恨，不然一个哨兵枪都造不出来就被拆了，造好哨兵枪后配合兵营出铁饼。

    # 美国头号威胁：战熊

    选帝国，爆 VX，还有一种打法是选苏联出电鱼。

    # 危险天空：维和轰炸机

    选帝国，爆天狗、VX，小心双刃拆家。

    # 无处可躲：间谍

    选帝国，爆步兵。

    # 展现武力：波能坦克

    选苏联，先用牛蛙带着工程师去偷后方的晋三，再用占领的晋三重工出海啸、天狗配合铁饼动员兵打掉另外两家。有几个要点：

    1. 牛蛙去晋三那的路上可能会被贤治的帽子打掉，可以开始出几头熊封兵营。
    2. 速度要快，等晋三出来天狗，工程师就被打死了，甚至刚从牛蛙里弹出来，还在天上就被拦截了。可以尝试用牛蛙主动跑过去扛伤害。
    3. 晋三每次重工的位置不同，但一般会出双重工，运气好的话，牛蛙可以在第二个重工建好前赶到。

    上面的方法容错率比较低，后面尝试出了简单一些的方法：选盟军，造兵营，出狗封右边的帝国，紧接着造工程师占领右边帝国兵营，出工程师占领两个矿，然后造帽子打掉这家。与此同时，家里攀科技（同时造些步兵守家），造先锋，用先锋打掉左边帝国和晋三。晋三出场后会打掉占领的三矿四矿，建议在造出两个先锋之后先派一个矿车去右下角开海里的矿做备份。还有一种更快地打法，就是工程师占领右边的同时，出步兵师攻打左边的帝国，但是比较考验操作。

    # 双面刃：海翼

    选帝国，出两个兵营，一个压制右侧，一个压制上侧，收基地去压右上角的矿车，可以打到两家出不来海翼，一家卖矿。后期适当出一些弓箭少女来对付海翼。飞行兵和流云的打法对我都不是很适用，前者会压制不住，导致对面海翼成型；后者会疯狂卡钱，没有足够的兵力出击。

    # 恶性循环：先锋武装战艇机

    选帝国，爆天狗和 VX。右上的电脑不像吉尔利斯，不怎么出飞机反而出 IFV 和 ACV。

    # 高水位线：航空母舰

    选帝国，开四矿，前期用小潜艇拿下左侧两个帝国的制海权，然后再爆海啸推掉两个帝国，此时盟军已经有了很多航母，不用怕，潜艇和海啸贴脸输出，航母就会不知所措，放 emp 的时候甚至还会误伤到自己人，

    # 奥米茄计划：百合子

    选帝国，爆天狗和百合子，天狗负责击杀对面的百合子，百合子负责拆家，一个百合子最多能举起四个天狗，所以天狗数量不能太少，尤其是推到最后一家的时候对面可能攒了相当数量的百合子，可以出一些武士来配合（分批，先来几个让百合子放 F）。

    第一次打的时候双重爆天狗+后期爆 VX 拆家也有机会在时间内完成。

    # 铁幕之后：铁幕装置

    选帝国，爆 VX，即使硬拆防空炮也能在时间内完成。

    也可以每次都用送的铁幕套在 VX 上再拆，套三次时间也来得及

    # 尽情收割：收割机甲

    选苏联，TR。

    # 万兽之王：V4火箭车

    最理想的情况是，选帝国，出一个迅雷去把将军刽子手引到盟军家里，然后冲基地到苏联，顺带爆海啸。但是要实现把将军刽子手顺利引过去是很难的，苏联出的镰刀也打不过，直接冲基地还可能被协议爆出的蜘蛛钻。

    目前成功的方案是选苏联，TR 苏联，用兵营、哨兵枪、围墙三个配合限制镰刀，占掉苏联后，一边出兵一边冲基地到盟军就可以了。

    # 雅典娜之怒：雅典娜战车

    选盟军，爆步兵，右侧的很快就能推掉，占他的矿，接着暴兵，MCV 也可以出动，去压雅典娜。

    # 准备出击：平定者

    选帝国，开局三矿，MCV 去压最左侧出来的平定者，然后爆 VX 和天狗。

    也可以只爆 VX 不爆天狗，但是注意一定要先把对面的阿波罗和机场打掉，不然容易团灭。

    # 不敌守护者：光谱塔

    选盟军，开局点高科协议，不用左下角的两个矿（会触发 AI 进攻），造完矿场后立马出一辆矿车跑到中间开矿，矿车展开需要时间，这段时间正好可以补个电厂，并点好一个矿场，展开后直接放三矿，然后造重工，提前收 MCV（不需要四矿），攒够四辆左右守护者就能出击了，MCV 冲在前面，至少能压掉对面一半的坦克，接下来 MCV 开到重工前面堵门。后面源源不断续守护者。

    一个常见的做法是先打掉对面的车库，但实际上意义并不大，因为 MCV 一下子就把对面坦克压死了，对面坦克能不能回血并不重要。

    还有一种打法是苏联起重开然后粉碎者+冲基地。

    # 拾荒者：娜塔莎

    选帝国，开局双兵营到中间，优先抢磁暴坦克、天启、鬼王，要点是磁暴坦克，因为磁暴坦克能反步兵，防止其他人出兵抢更多载具，只要前期压制住，就没问题了。

    # 你的金矿塌了：幻影坦克

    选盟军，爆步兵即可。

    这关后面打的时候重试了不下数十次，因为选盟军的话对手很喜欢造许多狗，然后步兵被狗的 F 消耗。后来参考流云的打法，先把基地拉到对手附近的水里，然后在对手门口造四兵营，才打过。

    # 生死之战：火箭天使

    选帝国，卖掉不需要的 T3、超武等，直接爆 VX。

    建议按照帝国、苏联、盟军的顺序拆。

    # 尽情杀戮：化学部队

    选帝国，出两个兵营开到对面门口，注意避让毒，暴兵即可。

    # 滨海杀戮：阿库拉潜艇

    选帝国，爆 VX，也可以选苏联爆电鱼。

    # 超时空：超时空传送仪

    选帝国，暴兵，家里多围几层围墙，或者 MCV 开到水里，防止超时空偷家。

    # 机器人与忍者：忍者

    选帝国，爆 VX 一定要围围墙，太容易被偷了。如果解锁了双刃的话用双刃应该也可以，还有爆电鱼的打法，但船厂容易被忍者钻。

    # 缩小地区：冷冻直升机

    选帝国，多出几个兵营核心，直接冲过去，因为纳米核心的大小是正常的，可以碾压其他载具、步兵、MCV。一个兵营展开出工程师占领其他建筑即可。

    # 破冰之战：冰冻军团

    选盟军，双兵营暴兵。

    注意避开冷冻协议，对面出了冷冻兵也不要怕，几个维和步兵贴脸喷即可，优先拆兵营和机场。

    # 大逃杀：剃刀巡洋舰

    选苏联，MCV开到中间占矿和油井，造点哨兵枪防御，出铁锤去进攻三家。

    # 海上对决：将军战舰

    选帝国，开局迅速爆小潜艇，绕过防御塔去打矿车，电脑就会一直卖。

    # 磁暴城堡：磁暴线圈

    选苏联，在左侧墙的另一侧造哨兵枪，可以打到磁暴坦克而不会被磁暴坦克打，开到左侧的矿，然后出 v4 去打大电，可配合娜塔莎。选盟军也可以，出平定者，不要用帝国，因为对面苏联有可能会把建筑建到城墙后，波能坦克要打的话需要绕一大圈，就超时了，而 v4 和平定者都可以隔着墙打。

    # 双管齐下：天启坦克

    选苏联，TR，围墙档对面的哨兵枪，在对面造兵营后造铁饼，优先拆重工。

    # 可恶的蒙古人：鬼王机甲

    选帝国，爆帽子，中间可以靠建筑干掉好几个鬼王，注意躲一下冲撞，不然会损失惨重，有医院可以回血，帽子数量起来后硬刚红鬼王也没问题。

    # 放马过来：超能波毁灭装置

    选苏联，起重开，直接 TR，用两个哨兵枪封住斜坡，这关的 AI 一定要把重工放到斜坡下面，所以重工核心会在路上被打爆，之后造重工出铁锤去打它的塔即可，也可以接着冲基地，应该能更快。

    # 军备竞赛：真空内爆弹

    这关也是个难点，对面发展很快，没过一会儿天启、娜塔莎、v4 就全出来了，而自己这边则不停卡钱，选盟军、帝国的话没有正面部队能打过对面，出冷冻也会被打下来，选苏联的话因为经济不如对面，所以哪怕出完全一样的东西也打不过，可能刚出来一辆天启就被娜塔莎狙掉了，而自己的娜塔莎还憋着出不来。目前的最佳方案是，选帝国，点自爆协议，速出天狗飞过去不停扰矿，虽然天狗很容易被对面的铁锤、铁饼打死，但这样能限制对方没钱出高科，也不会派兵过来，趁这个喘息的机会开出四矿，升 T3，用鬼王、百合子、火箭天使推掉对面。

    如果天狗不停扰矿（配合协议砸矿场）+拆掉了两个油井（可以每个油井提前放一个兵营造三个帽子），对手甚至会穷到卖重工，这时候随便用鬼王或天使处理敌人已有队伍（铁饼、瓶子、铁锤、收割、磁爆步兵）即可。

    注意不要占领中间的 车厂，否则四矿还没开完敌人就会出动军队攻打。

    # 最后的倒数：质子撞击炮

    选盟军，用平定者去轰，右边第一家的质子撞击跑可以占领成自己的。两个平定者就可以炸掉一家，优先炸机场，稳扎稳打，炸完一家再炸下一家时间也完全来得及。

    还有一个打法是帝国爆海啸，但在实操中容易被对面的多功能炮塔和光棱塔血虐，还不如爆 VX 和天狗。

    # 红色突袭：粉碎者

    选盟军，这关粉碎者有个 bug 是不会攻击海上的建筑，把基地车开过去，在海上放几座多功能炮塔即可。

    # 祖国：基洛夫空艇

    选帝国，家里多放围墙或炮塔，左边一个，右边一个，上边一个开防空打双刃，双重工，一个出天狗，一个出 VX，天狗不光可以用来打米格，也可以用来打来拆家的 V4，所以家里要留一些天狗，VX 不擅长对付的铁饼也可以用天狗来解决，注意观察防空炮。

    # 超级指挥官：无畏级战舰

    选帝国，敌军只会从上方盟军和下方苏联的方向过来，这两个地方多摆一些炮台，苏联负责供电和出米格，帝国把矿开满，盟军攀科技出先锋，帝国出兵建筑也摆在盟军那，出鬼王、百合子、火箭天使，光是地上部队基本就可以把上方的两个帝国推平，先锋够三个后去打苏联，用米格掩护。

    # 战斗室：纳米虫巢

    选盟军，最下面的基地直接卖掉，中间的基地收起来开路，上方基地双兵营暴兵，敌方兵大都是排成直线的，MCV 直接碾过去。

    还有娱乐打法，最下面基地卖掉，最上方基地开到右下角海里开矿和攀科技，接下来如果选的是盟军可以出冷冻军团，苏联则是出辐射兵。风险比较大，中间还容易被拆家，速度也慢，造出的高级兵种还可能被武士砍死，仅供娱乐。

    # 拨云见日：世纪轰炸机

    选苏联，双兵营爆火炮机车，家里多围围墙或放个炮台防伞兵。

    为了防止基地车跑到水里，可以提前在敌人南方的水域放一个恐怖机器人。

    # 帝国之怒：超级要塞

    选帝国，双兵营爆火箭天使，对面的超级要塞直接被瘫痪，配合百合子拆家。

    # 未来战争：未来坦克

    选苏联，家里多造防空炮，左上角未来坦克去左上角基地，右下角未来坦克去右下角基地，从高地对着兵营重工强 A，等双刃造好后由双刃把未来坦克带下去以规避光谱塔。剩下的左下角和右上角直接用双刃把未来坦克载过去即可，期间一定要造米格把对面的冷冻打下来。
